Proyecto POS Virtual

Objetivo

Desarrollar una aplicación web para punto de venta (POS) virtual, dirigida a pequeños y medianos negocios, permitiendo procesar pagos mediante tarjetas con tecnología NFC o a través de transferencias bancarias vía código QR.

Características Principales

  • Interfaz amigable y adaptable a dispositivos móviles.
  • Autenticación de usuarios con roles (admin, vendedor).
  • Registro de ventas y generación de tickets digitales.
  • Integración con pagos NFC (tarjetas de débito/crédito).
  • Generación automática de código QR con datos del negocio para recibir transferencias.
  • Historial de transacciones por cliente o por fecha.

Tecnologías a Usar

  • Frontend: React + TailwindCSS
  • Backend: Node.js + Express
  • Base de datos: MongoDB
  • Autenticación: JWT
  • API externa de bancos para validación de transferencias

Futuras mejoras

  • Implementación de facturación electrónica integrada.
  • Soporte para varios dispositivos por usuario.
  • Módulo de inventario sincronizado.
  • Soporte offline y sincronización automática.

Estado actual

En fase de planificación y validación de APIs bancarias disponibles. Se están definiendo los flujos principales y evaluando compatibilidad con dispositivos para NFC.
